

# Sistema de Orçamentos da Construção Civil

### 📌 Descrição
Sistema desenvolvido para cálculo de orçamentos em obras de construção civil, com suporte a serviços como alvenaria, reboco, porcelanato, cerâmica, concreto e gesso.

### 🚀 Funcionalidades
- Cálculo de medidas em m² e m³
- Geração de PDF com dados do cliente e resultados
- Interface responsiva para uso em dispositivos móveis
