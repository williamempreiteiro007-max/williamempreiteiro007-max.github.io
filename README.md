# Sistema de Orçamentos da Construção Civil


### 🚀 Descrição
Sistema desenvolvido para cálculo de orçamentos em obras de construção civil, com suporte a serviços como alvenaria, reboco, porcelanato, cerâmica, concreto e gesso. Funciona diretamente no navegador e é otimizado para celulares Android.


### ✨ Funcionalidades
- Cálculo de medidas em m² (área) e m³ (volume)
- Geração de PDF customizado com dados do cliente e resultados dos cálculos
- Interface responsiva com botões arredondados e design chamativo
- Tela inicial com imagem do leão como visual principal
- Opção de alterar valor total do orçamento antes de gerar o PDF


### 📂 Arquivos do Projeto
- `leao (1).jpg`: Imagem de visualização na tela inicial
- `index.html`: Estrutura completa da interface, lógica em JavaScript e ligação com bibliotecas
- `style.css`: Estilização da interface (responsividade, cores, botões)
- `README.md`: Instruções e informações sobre o sistema


### 📱 Como Usar
1. Acesse o sistema pelo link: https://williamempreiteiro007-max.github.io
2. Na tela inicial, clique em **"COMEÇAR"**
3. Selecione o tipo de serviço desejado
4. Insira as medidas solicitadas e clique em **"CALCULAR"**
5. Confira o resultado, preencha os dados do cliente e escolha se deseja incluir os cálculos no PDF
6. Clique em **"GERAR PDF"** para salvar o arquivo


### 🛠️ Tecnologias Utilizadas
- HTML5 para estrutura
- CSS3 para estilização
- JavaScript para lógica e cálculos
- Biblioteca jsPDF para geração de PDFs
- GitHub Pages para hospedagem online
